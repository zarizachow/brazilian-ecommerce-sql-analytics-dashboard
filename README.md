# Brazilian E-Commerce SQL Analytics Dashboard

## Project Overview

This project presents an end-to-end business intelligence analysis of the Brazilian Olist e-commerce dataset. The objective is to uncover actionable insights into customer behavior, revenue performance, product categories, and operational efficiency using SQL and interactive dashboards.

The project emphasizes structured data modeling, analytical SQL queries, and executive-ready visualizations.

---

## Objectives

- Analyze revenue trends and growth patterns  
- Evaluate customer purchasing behavior  
- Assess delivery performance and operational metrics  
- Identify top-performing product categories  
- Explore payment patterns and review scores  

---

## Tools & Technologies

- SQL (data modeling, cleaning, analysis)  
- Power BI / Tableau (interactive dashboards)  
- Git & GitHub (version control)
